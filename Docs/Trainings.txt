При нажатии на кнопку "Trainings" в боковом меню появится вкладка с информацией о ближайшей записи на тренировку для конкретного пользователя, которого был выполнен вход. Боковое меню будет зафиксировано.

Также на вкладке присутствуют кнопки с каждым из трениров, при нажатии на котороые ниже высветится расписание на ближайшие рабочие дни для выбранного тренера. После будет необходимо выбрать тип треинровки, нажатием на кнопку "Pool"/"Strength"/"Cardio". Если вид тренировки не был выбран, то автоматически будет поставлен тип "Strength". Далее при просмотре расписания пользователь может выбрать время и дату, а после написать их в появившиеся поля для даты и времени. После введения всех полей для записи на тренировку необходимо нажать на кноку "Sign up". Тогда осуществится проверка, если введённые данные неверны, то ничего не произойдёт. Если введённые данные верны, то из расписания выбранного тренера будет удалено данное время, так как оно теперь является занятым.
