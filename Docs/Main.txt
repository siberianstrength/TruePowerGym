В главном меню есть личная информация о пользователе, которую можно редактировать, нажав на кнопку с иконкой карандаша. После осуществляется ввод информации и подтверждение. Для обновления информации о птользователе необходио нажать на круглую стрелку. Тогда изменённая информация будет обновлена.

Вглавном меню по центру сверху присутствует кнопка формы здоровья в виде листов бумаги. Там присутствует информация о состоянии здоровья клиента, для которого была введена информация при авторизации. Слева присутствует кнопка добавления заметки при изменении здоровья пользователя. После добавления заметки необходимо нажать кнопку подтверждения, расположенную по центру сниза экрана. После подтвержения нужно нажать круглую стрелку для обновления информации, после чего новая информация появится на экране.

Для возвращения в главное меню необходимо нажать на круглую кнопку с изображением человека.

Также в главном меню присутствуют 4 карточки-абонемента, которые яляются кликабельными кнопками, при нажатии на которые выводится информация о доступных видах занятий и дополнительных возможностях, включённых именно в данный тип абонемента. Для закрытия страницы нужно нажать крестик справа сверху.

В главном меню выводится тип абонемента текущего пользователя. Является не кликабельным.

Если увести мышку влево экрана, появится бокове меню.
